# List controls

{numref}`list_controls` 这些 SESSION 命令用于控制列表和打印程序输出。

```{table} List controls commands
:name: list_controls

| | | |
|---|---|---|
| {doc}`Mapdl.com([comment]) <../204_List_controls/com>` |Places a comment in the output.| 在输出中放置一个注释。 |
| {doc}`Mapdl.golist(kwargs) <../204_List_controls/golist>` | Reactivates the suppressed data input listing. |重新激活被抑制的数据输入列表。|
| {doc}`Mapdl.gopr(kwargs) <../204_List_controls/gopr>` |Reactivates suppressed printout.| 重新激活被抑制的打印输出。 |
| {doc}`Mapdl.nolist(kwargs) <../204_List_controls/nolist>` |Suppresses the data input listing.| 抑制数据输入列表。 |
| {doc}`Mapdl.nopr(kwargs) <../204_List_controls/nopr>` | Suppresses the expanded interpreted input data listing. |取消扩展的解释输入数据列表。|

```
