# aux2

```{py:method} Mapdl.aux2(**kwargs)

Enters the binary file dumping processor.\
进入二进制文件转储处理器。

APDL Command: `/AUX2`

Notes
----------

Enters the binary file dumping processor (ANSYS auxiliary processor AUX2). This processor is used to dump the contents of certain ANSYS binary files for visual examination.\
进入二进制文件转储处理器（ANSYS 辅助处理器 AUX2）。这个处理器用于转储某些 ANSYS 二进制文件的内容，以便进行可视化检查。

This command is valid only at the Begin Level.\
这个命令仅在初始层有效。

```
