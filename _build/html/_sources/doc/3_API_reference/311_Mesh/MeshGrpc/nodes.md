# nodes

````{property} property MeshGrpc.nodes: ndarray

Array of nodes in Global Cartesian coordinate system.

Examples
--------

```python
>>> mapdl.mesh.nodes
array([[0.   0.   0.  ]
       [1.   0.   0.  ]
       [0.25 0.   0.  ]
       ...,
       [0.75 0.5  3.5 ]
       [0.75 0.5  4.  ]
       [0.75 0.5  4.5 ]]
```
````
