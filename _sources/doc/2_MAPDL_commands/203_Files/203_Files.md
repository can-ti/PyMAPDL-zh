# Files

{numref}`files` 这些 SESSION 命令用于文件操作，例如删除、复制和列表。

```{table} Files commands
:name: files

| | | |
|---|---|---|
| {doc}`Mapdl.anstoaqwa([fname,vertaxis,gc,rho,...]) <../203_Files/anstoaqwa>` |Creates an AQWA-LINE input file from the current ANSYS model.| 从当前的 ANSYS 模型创建一个 AQWA-LINE 输入文件。|
| {doc}`Mapdl.anstoasas([fname,key]) <../203_Files/anstoasas>` |Creates an AQWA-LINE input file from the current ANSYS model.| 从当前的 ANSYS 模型创建 ASAS 输入文件。 |
| {doc}`Mapdl.assign([ident,fname,ext,lgkey]) <../203_Files/assign>` |Reassigns a file name to an ANSYS file identifier.| 将文件名重新分配给 ANSYS 文件标识符。 |
| {doc}`Mapdl.copy([fname1,ext1,fname2,ext2,distkey]) <../203_Files/copy>` |Copies a file.| 复制文件。 |
| {doc}`Mapdl.slashdelete([fname,ext,diskey]) <../203_Files/slashdelete>` |Deletes a file.| 删除文件。 |
| {doc}`Mapdl.fcomp([ident,level]) <../203_Files/fcomp>` |Specifies file compression level.| 指定文件压缩级别。 |
| {doc}`Mapdl.lgwrite([fname,ext,kedit,...]) <../203_Files/lgwrite>` |Writes the database command log to a file.| 将数据库命令日志写入文件。 |
| {doc}`Mapdl.starlist([fname,ext]) <../203_Files/starlist>` |Displays the contents of an external, coded file.| 显示外部编码文件的内容。 |
| {doc}`Mapdl.slashclog([fname,ext]) <../203_Files/slashclog>` |APDL Command: /CLOG| APDL命令:/CLOG |
| {doc}`Mapdl.slashfdele([ident,stat]) <../203_Files/slashfdele>` |Deletes a binary file after it is used.| 在一个二进制文件被使用后将其删除。 |
| {doc}`Mapdl.rename([fname,ext1,fname2,ext2,...]) <../203_Files/rename>` |Renames a file.| 重命名文件。 |

```
