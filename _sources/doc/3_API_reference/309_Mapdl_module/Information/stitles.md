# stitles

```{property} property Information.stitles

Retrieve or set the value for the MAPDL stitle (subtitles).\
读取或设置 MAPDL stitle（suntitles）的值。

If ‘stitle’ includes newline characters ('/n'), then each line is assigned to one STITLE.\
如果 "stitle "包含换行符 ('/n')，则每行分配一个 STITLE。

If ‘stitle’ is equals `None`, the stitles are reset.\
如果 "stitle" 等于 "None"，字幕将被重置。

If `i` is supplied, only set the stitle number i.\
如果提供了 `i`，则只设置标题编号 i。

Starting from 0 up to 3 (Python indexing).\
从 0 到 3（Python 索引）。


```
