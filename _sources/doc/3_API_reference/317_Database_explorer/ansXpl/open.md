# open

````{method} ansXpl.open(filename, option='')

Open an MAPDL file to explore.

Parameters:
---------

  filename : *`str`*
  : Name of the file to open.

Returns:
---------

  `str`
  : Response from MAPDL.

Examples
---------

```python
>>> xpl.open('file.mode')
===============================================
=====      ANSYS File Xplorer            ======
````
