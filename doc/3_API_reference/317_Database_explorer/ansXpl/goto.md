# goto

````{method} ansXpl.goto(path)

Go directly to a new location in the file.

Parameters:
---------

  path : *`str`*
  : Absolute path to the new location.\
  新位置的绝对路径。


Examples
---------

```python
>>> print(xpl.goto('MASS'))
 =====      ANSYS File Xplorer : Go up to top level(s)
 =====      ANSYS File Xplorer : Step into Block MASS
```
````
