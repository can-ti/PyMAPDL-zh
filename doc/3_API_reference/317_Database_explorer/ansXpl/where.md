# where

````{method} ansXpl.where()

Returns the current location in the MAPDL file.\
返回 MAPDL 文件中的当前位置。


Returns:
-------
  `str`
  : String containing the current location.\
  包含当前位置的字符串。

Examples
-------

```python
>>> print(xpl.where())
 =====      ANSYS File Xplorer : Display Current Location
 Current Location : FULL
    File Location : 412
```
````
