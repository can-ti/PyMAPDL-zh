# Plotting

Various PyMAPDL specific plotting commands.\
各种 PyMAPDL 专用绘图命令。

```{table}

| | | |
|---|---|---|
| {doc}`plotting.general_plotter(meshes, points, ...) <../313_Plotting/general_plotter>` | General pymapdl plotter for APDL geometry and meshes. | 用于 APDL 几何图形和网格的通用 pymapdl 绘图。 |
| {doc}`Mapdl.aplot([na1, na2, ninc, degen, scale, ...]) <../313_Plotting/aplot>` | Display the selected areas. | 显示所选面图素。 |
| {doc}`Mapdl.eplot([show_node_numbering, vtk]) <../313_Plotting/eplot>` | Plots the currently selected elements. | 绘制当前选中的元素。 |
| {doc}`Mapdl.kplot([np1, np2, ninc, lab, vtk, ...]) <../313_Plotting/kplot>` | Display the selected keypoints. | 显示选定的关键点。 |
| {doc}`Mapdl.lplot([nl1, nl2, ninc, vtk, ...]) <../313_Plotting/lplot>` | Display the selected lines. | 显示所选线图素。 |
| {doc}`Mapdl.nplot([nnum, vtk]) <../313_Plotting/nplot>` | APDL Command: `NPLOT` |  |
| {doc}`Mapdl.vplot([nv1, nv2, ninc, degen, scale, ...]) <../313_Plotting/vplot>` | Plot the selected volumes. | 绘制所选体图素。 |

```
